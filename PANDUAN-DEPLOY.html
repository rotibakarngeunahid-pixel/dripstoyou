Beli Server VPS

Analoginya: VPS itu seperti menyewa komputer di tempat lain yang nyala 24 jam. Website Anda "tinggal" di sana dan bisa diakses siapapun dari seluruh dunia.

Kami merekomendasikan salah satu dari provider berikut (harga terjangkau, kualitas bagus):

ProviderPaketEstimasi HargaLink
DigitalOcean Rekomendasi Droplet Basic — 1 vCPU / 2GB RAM / 50GB $12/bulan (~Rp 190rb) digitalocean.com
Vultr Cloud Compute — 1 vCPU / 2GB RAM $10/bulan (~Rp 160rb) vultr.com
Hostinger VPS VPS 1 — 1 vCPU / 4GB RAM Rp 90rb–150rb/bulan hostinger.co.id

Yang perlu dipilih saat order:

  • Sistem operasi: pilih Ubuntu 22.04 LTS (wajib)
  • Lokasi server: pilih Singapore (paling dekat dengan Bali)
  • Aktifkan SSH key (atau biarkan pakai password, lebih mudah untuk pemula)
  • Setelah bayar, Anda akan mendapat IP Address server (contoh: 103.45.67.89) — catat!
💡 Setelah server aktif, arahkan domain Anda ke IP server tersebut di pengaturan DNS domain Anda. Contoh: buat record A untuk dripstoyou.com → 103.45.67.89. Proses ini bisa makan waktu 1–24 jam (propagasi DNS).

Masuk ke Server (SSH)

Analoginya: SSH itu seperti "remote desktop" tapi pakai perintah teks. Anda mengetik perintah di komputer Anda, tapi yang dijalankan adalah komputer server di Singapore.

Cara masuk dari Windows:

  • Buka aplikasi Terminal atau PowerShell (tekan tombol Windows, ketik "Terminal", klik Open)
  • Ketik perintah ini, ganti IP_SERVER dengan IP yang Anda dapat dari provider:
ssh root@IP_SERVER # Contoh: ssh root@103.45.67.89
  • Ketik yes jika muncul pertanyaan "Are you sure you want to continue connecting?"
  • Masukkan password server (yang dikirim via email dari provider)
  • Jika berhasil, layar akan menampilkan teks seperti root@server:~# — artinya Anda sudah masuk!

Cara masuk dari Mac:

Sama saja — buka Terminal (Finder → Applications → Utilities → Terminal), lalu ketik perintah SSH yang sama di atas.

ℹ️ Saat mengetik password di terminal, tidak akan muncul tanda apapun (bintang atau titik). Ini normal! Tetap ketik passwordnya, lalu tekan Enter.

Install Software yang Dibutuhkan

Setelah masuk ke server, copy-paste perintah berikut satu per satu. Tunggu sampai selesai sebelum mengetik perintah berikutnya.

3a. Update sistem (wajib dilakukan pertama kali)

apt update && apt upgrade -y

⏱ Tunggu 2–5 menit hingga selesai.

3b. Install Node.js 20 (mesin yang menjalankan website)

curl -fsSL https://deb.nodesource.com/setup_20.x | bash - apt install -y nodejs

Verifikasi berhasil:

node --version # Harus muncul: v20.x.x

3c. Install MySQL 8 (database — tempat menyimpan data)

apt install -y mysql-server mysql_secure_installation

Saat muncul pertanyaan, jawab seperti ini:

PertanyaanJawaban
Would you like to setup VALIDATE PASSWORD component?N
Change the password for root?Y → masukkan password kuat (catat!)
Remove anonymous users?Y
Disallow root login remotely?Y
Remove test database?Y
Reload privilege tables?Y

3d. Install Nginx (pintu masuk website dari internet)

apt install -y nginx systemctl start nginx systemctl enable nginx

3e. Install PM2 (menjaga website tetap nyala 24 jam)

npm install -g pm2
✅ Semua software sudah terinstall! Sekarang server Anda siap untuk menjalankan website.

Upload File Website ke Server

Opsi A — Menggunakan Git (direkomendasikan jika ada repository)

Jika kode website disimpan di GitHub, ini caranya:

# Buat folder untuk website mkdir -p /var/www/dripstoyou cd /var/www/dripstoyou # Clone dari GitHub (ganti URL dengan repo Anda) git clone https://github.com/NAMA_AKUN/dripstoyou-bali.git . # Install dependencies npm install

Opsi B — Upload Manual via FileZilla (jika tidak pakai GitHub)

  • Download FileZilla (gratis) di filezilla-project.org
  • Buka FileZilla → File → Site Manager → New Site
  • Isi:
    • Protocol: SFTP
    • Host: IP server Anda
    • User: root
    • Password: password server Anda
  • Klik Connect
  • Di panel kanan (server), navigasi ke folder /var/www/ → klik kanan → buat folder baru bernama dripstoyou
  • Di panel kiri (komputer Anda), buka folder project DRIP TO YOU
  • Pilih semua file → drag ke panel kanan (server)
  • Tunggu upload selesai (tergantung kecepatan internet, ~5–15 menit)

Setup Database

Analoginya: Database itu seperti lemari arsip digital. Di sinilah semua data booking, produk, dan admin Anda tersimpan.

Buat database baru

Masuk ke MySQL, ganti PASSWORD_ROOT dengan password MySQL yang Anda buat tadi:

mysql -u root -p

Masukkan password, lalu ketik perintah-perintah ini satu per satu (termasuk titik koma di akhir!):

CREATE DATABASE dripstoyou CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci; CREATE USER 'drip'@'localhost' IDENTIFIED BY 'GantiPasswordIni123!'; GRANT ALL PRIVILEGES ON dripstoyou.* TO 'drip'@'localhost'; FLUSH PRIVILEGES; EXIT;
⚠️ Ganti GantiPasswordIni123! dengan password kuat pilihan Anda (min. 12 karakter, ada huruf besar, angka, simbol). Catat password ini! Anda akan butuhnya di langkah berikutnya.

Isi File Konfigurasi (.env)

Analoginya: File .env itu seperti "kartu pengaturan" untuk website Anda — berisi info penting seperti password database, nomor WhatsApp, dll. File ini rahasia dan tidak boleh dibagikan.

Buat file .env

Di terminal server, masuk ke folder website dan buat file .env:

cd /var/www/dripstoyou cp .env.example .env nano .env

Ini akan membuka editor teks di terminal. Anda bisa navigasi dengan tombol panah, dan edit langsung.

Isi setiap baris dengan nilai yang benar:

Nama Variabel Contoh Nilai Penjelasan
DATABASE_URL "mysql://drip:Password123!@localhost:3306/dripstoyou" Ganti Password123! dengan password DB Anda (langkah 5)
SESSION_SECRET "(hasil dari perintah di bawah)" String acak untuk keamanan sesi login admin
FIELD_ENCRYPTION_KEY "(hasil dari perintah di bawah)" Kunci enkripsi data pelanggan (no HP, alamat)
WHATSAPP_NUMBER "6281234567890" Nomor WA bisnis tanpa + atau spasi, awali 62
NEXT_PUBLIC_WHATSAPP_NUMBER "6281234567890" Sama persis dengan WHATSAPP_NUMBER di atas
NEXT_PUBLIC_APP_URL "https://dripstoyou.com" Domain website Anda (dengan https://)
NODE_ENV "production" Jangan diubah

Generate SESSION_SECRET dan FIELD_ENCRYPTION_KEY:

Buka terminal kedua (atau pause nano dengan Ctrl+Z), lalu jalankan:

# Generate SESSION_SECRET (min 32 karakter) openssl rand -base64 32 # Contoh output: xK8mP2nQ9rL4wX7vZ1oE5jY3aH6cB0dF+sW== # Generate FIELD_ENCRYPTION_KEY (harus tepat 64 karakter hex) openssl rand -hex 32 # Contoh output: a3f9b2e1d4c7a0b8e5f2d9c6a1b4e7f0a3f9b2e1d4c7a0b8e5f2d9c6a1b4e7f0

Copy hasilnya, lalu paste ke file .env.

Simpan file .env:

Tekan Ctrl + X → ketik Y → tekan Enter

🔴 SANGAT PENTING: Jangan pernah membagikan isi file .env ke siapapun — termasuk via chat, email, atau screenshot. File ini berisi kunci rahasia yang menjaga keamanan seluruh data bisnis Anda.

Build dan Jalankan Website

Ini bagian di mana website "dikompilasi" dan dihidupkan. Jalankan perintah-perintah berikut secara berurutan dari folder website:

7a. Masuk ke folder website

cd /var/www/dripstoyou

7b. Generate Prisma (siapkan koneksi database)

npx prisma generate

⏱ Tunggu hingga muncul pesan "✔ Generated Prisma Client"

7c. Jalankan migrasi database (buat semua tabel)

npx prisma migrate deploy

⏱ Tunggu hingga muncul pesan "All migrations have been successfully applied"

ℹ️ Jika muncul error soal database, periksa kembali DATABASE_URL di file .env — pastikan password dan nama database sudah benar.

7d. Isi data awal (products, area, admin)

npm run db:seed

⏱ Tunggu hingga muncul pesan "✅ Seed complete!"

7e. Build website (proses terlama, ~5–10 menit)

npm run build

⏱ Tunggu hingga muncul tabel route dan pesan berhasil. Jangan tutup terminal!

7f. Jalankan website dengan PM2

pm2 start npm --name "dripstoyou" -- start pm2 save pm2 startup

Setelah pm2 startup, akan muncul perintah panjang yang perlu di-copy dan dijalankan. Contohnya seperti ini (jalankan perintah yang muncul, bukan contoh ini):

# Contoh perintah yang perlu dijalankan (akan berbeda di server Anda): sudo env PATH=$PATH:/usr/bin /usr/lib/node_modules/pm2/bin/pm2 startup systemd -u root --hp /root

Cek apakah website sudah berjalan:

pm2 status # Harus muncul: dripstoyou | online

Setup Domain dan SSL (HTTPS)

Analoginya: Nginx itu seperti "resepsionis" yang menerima tamu (pengunjung website) dan mengarahkan mereka ke dalam. SSL adalah gembok yang membuat koneksi aman (lambang 🔒 di browser).

8a. Buat file konfigurasi Nginx

nano /etc/nginx/sites-available/dripstoyou.com

Copy-paste seluruh teks di bawah ini ke editor (ganti dripstoyou.com dengan domain Anda jika berbeda):

server { listen 80; server_name dripstoyou.com www.dripstoyou.com; return 301 https://$host$request_uri; } server { listen 443 ssl http2; server_name dripstoyou.com www.dripstoyou.com; ssl_certificate /etc/letsencrypt/live/dripstoyou.com/fullchain.pem; ssl_certificate_key /etc/letsencrypt/live/dripstoyou.com/privkey.pem; location / { proxy_pass http://127.0.0.1:3000; proxy_http_version 1.1; proxy_set_header Host $host; proxy_set_header X-Real-IP $remote_addr; proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for; proxy_set_header X-Forwarded-Proto $scheme; } }

Simpan: Ctrl+X → Y → Enter

8b. Aktifkan konfigurasi Nginx

ln -s /etc/nginx/sites-available/dripstoyou.com /etc/nginx/sites-enabled/ nginx -t # Harus muncul: configuration file test is successful systemctl reload nginx

8c. Install SSL gratis dengan Certbot

apt install -y certbot python3-certbot-nginx certbot --nginx -d dripstoyou.com -d www.dripstoyou.com

Saat diminta, masukkan email Anda (untuk notifikasi expiry), setujui Terms of Service (A), pilih tidak untuk berbagi email (N).

✅ Jika berhasil, Certbot akan otomatis update file Nginx dan SSL aktif. Sertifikat SSL gratis dan auto-renew setiap 90 hari.
⚠️ SSL hanya bisa dibuat jika domain sudah mengarah ke IP server (propagasi DNS sudah selesai). Jika error, tunggu beberapa jam dan coba lagi.

Checklist Setelah Deploy

Setelah selesai deploy, centang semua item berikut:

Website Publik

  • Homepage terbuka di https://dripstoyou.com tanpa error
  • Halaman Treatments menampilkan 4 produk
  • Form Booking bisa diisi dan tombol submit berfungsi
  • Link WhatsApp di homepage dan booking mengarah ke nomor yang benar
  • Halaman FAQ menampilkan pertanyaan
  • Ikon gembok 🔒 muncul di browser (SSL aktif)

Panel Admin

  • Login admin berhasil di /admin/login
  • Dashboard menampilkan statistik booking
  • Menu Produk menampilkan 4 treatment yang sudah diseed
  • Menu Jadwal bisa diedit dan disimpan
  • Pengaturan: nomor WhatsApp sudah benar
  • Password admin sudah diganti dari default!

Keamanan

  • File .env tidak bisa diakses dari browser (coba buka https://dripstoyou.com/.env → harus 404 atau error)
  • HTTP redirect ke HTTPS (coba buka http://dripstoyou.com → harus redirect)
  • Backup file .env disimpan di tempat aman (bukan di server)
