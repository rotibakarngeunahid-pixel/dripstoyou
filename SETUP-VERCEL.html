🚀 Setup Website DRIP TO YOU

3 langkah saja — total waktu sekitar 5 menit

1
Aktifkan Remote MySQL di Rumahweb
⏱ 1 menit
⚠️ Wajib dilakukan pertama! Tanpa ini, Vercel tidak bisa menyambung ke database Anda.
1
Login ke cPanel Rumahweb Anda
2
Di kolom pencarian cPanel, ketik "Remote MySQL" lalu klik
3
Di kolom "Host", ketik karakter persen: %
4
Klik tombol "Add Host" — selesai!
ℹ️ Simbol % artinya "izinkan dari semua IP" — diperlukan karena Vercel pakai banyak IP berbeda.
2
Isi Environment Variables di Vercel
⏱ 3 menit
1
Buka vercel.com → klik project dripstoyou
2
Klik tab Settings → di menu kiri klik Environment Variables
3
Tambahkan 7 variable berikut satu per satu. Klik tombol Copy lalu paste ke kolom Value:

4
Setiap variable: isi Name dan Value → pastikan centang Production → klik Save
3
Klik Redeploy di Vercel
⏱ 5 menit (otomatis)
1
Di Vercel, klik tab Deployments
2
Pada baris paling atas (deployment terbaru), klik titik tiga ⋯
3
Klik "Redeploy" → konfirmasi → tunggu
4
Tunggu sampai status berubah jadi ● Ready (hijau) — sekitar 3-5 menit
✅ Saat Redeploy, Vercel otomatis: buat semua tabel database + isi data awal (produk, area, jadwal, admin). Tidak perlu langkah tambahan!
